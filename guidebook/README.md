# Hermes 自我进化 guidebook

这套 guidebook 不打算先按“通用 agent 项目分层”去介绍 Hermes。

它更关心的是这条主线：

> **Hermes 怎样把一次次对话中的经验，沉淀成下一次运行可以直接使用的长期能力。**

也就是说，这套书最值得先看的，不是 `run_agent.py` 的 loop 本身，而是：

- 持久记忆怎样被稳定注入
- 会话回忆怎样把过去重构成当前可用经验
- skill system 怎样把成功做法沉淀成 procedural memory
- compression / caching / background review 怎样让经验继续变轻、变稳、变可复用
